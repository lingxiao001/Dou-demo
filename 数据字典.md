# 数据字典

本文档定义了项目中使用的核心数据实体及其字段。

---

## 1. User (用户)

代表一个独立的用户账户。

| 字段名 | 数据类型 | 描述 | 示例 |
| :--- | :--- | :--- | :--- |
| `id` | `String` | 用户的唯一标识符。 | `"user1"` |
| `nickname` | `String` | 用户显示的昵称。 | `"Flutter开发者"` |
| `avatarUrl` | `String` | 用户头像图片的 URL。 | `"https://randomuser.me/api/portraits/men/1.jpg"` |

---

## 2. VideoPost (视频帖子)

代表一个发布的视频内容，是信息流中的核心对象。

| 字段名 | 数据类型 | 描述 | 示例 |
| :--- | :--- | :--- | :--- |
| `id` | `String` | 视频帖子的唯一标识符。 | `"1"` |
| `coverUrl` | `String` | 视频封面图片的 URL。 | `"https://img.youtube.com/vi/pBFt_y6gJkg/0.jpg"` |
| `videoUrl` | `String` | 视频播放地址的 URL。 | `"https://www.youtube.com/watch?v=pBFt_y6gJkg"` |
| `title` | `String` | 视频的标题或描述文字。 | `"探索 Flutter 的世界"` |
| `likeCount` | `int` | 点赞数量。 | `12000` |
| `commentCount` | `int` | 评论数量。 | `845` |
| `author` | `User` | 发布该视频的作者信息。 | `User(...)` |
| `musicTitle` | `String` | 视频所用背景音乐的标题。 | `"Flutter 之歌"` |
| `musicCoverUrl` | `String` | 背景音乐的封面图片 URL。 | `"https://picsum.photos/seed/music1/100"` |
| `isLiked` | `bool` | 当前用户是否已点赞该视频。 | `false` |
| `createdAt` | `DateTime` | 视频的发布时间。 | `DateTime.parse(...)` |

---

## 3. Comment (评论)

代表针对某个视频帖子的一条评论。

| 字段名 | 数据类型 | 描述 | 示例 |
| :--- | :--- | :--- | :--- |
| `id` | `String` | 评论的唯一标识符。 | `"comment1"` |
| `postId` | `String` | 该评论所属视频帖子的 ID。 | `"1"` |
| `author` | `User` | 发表该评论的用户信息。 | `User(...)` |
| `content` | `String` | 评论的具体内容。 | `"讲得太好了！"` |
| `createdAt` | `DateTime` | 评论的发布时间。 | `DateTime.parse(...)` |
